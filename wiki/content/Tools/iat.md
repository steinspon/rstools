# Image Annotating Tool (IAT)

IAT is used to calibrate image scale and place clear technical callouts.

## Suggested workflow

1. Load source image.
2. Calibrate scale using a known reference.
3. Add annotations, arrows, and labels.
4. Export publication-ready visuals.

## Good practice

- Prefer concise labels over long paragraphs.
- Use consistent style when preparing multi-image reports.
